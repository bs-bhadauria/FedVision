# FedVision - Official Landing Page

![FedVision Banner](https://img.shields.io/badge/FedVision-Privacy%20First%20Surveillance-blue?style=for-the-badge)

This repository contains the frontend source code for the official promotional landing page of **FedVision**.

FedVision is a cutting-edge, 5G-enabled active surveillance platform powered by **Federated Learning**. It allows for real-time video analytics and model training at the edge without compromising data privacy.

## Confidentiality Notice

**Please Note:** This repository *only* hosts the static assets (HTML, CSS, JavaScript) for the public-facing landing page. 

The core intellectual property—including the proprietary Federated Learning orchestrator, computer vision models, video ingestion pipelines, and edge-node communication protocols—is maintained in a strictly **private** and secure environment. It is not publicly available.

## Tech Stack

This landing page is designed to be blazing fast, secure, and lightweight:
- **HTML5:** Semantic structuring.
- **CSS3:** Custom styling, modern glassmorphism UI, and smooth keyframe animations (No external heavy CSS frameworks used).
- **JavaScript (Vanilla):** Dynamic interactions, ROI calculator, and SVG animation sequencing.
- **Web3Forms:** Serverless contact form integration for pilot project inquiries.

## Hosting & Deployment

This static site is optimized for deployment on **GitHub Pages**.

1. Commit all files (`index.html`, `style.css`, `main.js`, `README.md`) to the `main` branch.
2. Go to Repository **Settings** > **Pages**.
3. Under "Build and deployment", select the `main` branch and `/root` folder.
4. Click **Save**. The website will be live in a few minutes!
